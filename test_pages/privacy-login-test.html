🔒 隱私承諾功能測試

功能說明

我們為社交登入添加了隱私承諾提示，讓用戶了解我們如何保護他們的資料安全。

✨ 新增功能

  • Google 登入隱私承諾：點擊 Google 登入時會顯示隱私承諾對話框
  • Facebook 登入隱私承諾：點擊 Facebook 登入時會顯示隱私承諾對話框
  • 美觀的模態框設計：使用現代化的 UI 設計，提升用戶體驗
  • 明確的承諾內容：清楚說明資料使用方式和安全保護措施
  • 用戶選擇權：用戶可以選擇同意或取消

隱私承諾內容

我們向用戶承諾：

  • 只取得基本資料（姓名、電子郵件）
  • 使用 Google Firebase 安全儲存，符合國際安全標準
  • 絕不會將資料出售或分享給第三方
  • 所有資料傳輸都經過 HTTPS 加密保護
  • 用戶可隨時在會員中心管理資料
